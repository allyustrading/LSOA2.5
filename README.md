# Liuli Society — Alternative Theme Website

A second GitHub-ready React + Vite version of the Liuli Society website with a completely different design direction.

## Theme direction
- dark editorial nonprofit style
- warmer serif headline feel
- museum / institution inspired layout
- image-led sections on every page
- nonprofit-safe content language

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Project structure
- `src/components/` reusable layout and UI
- `src/pages/` page content
- `src/assets/` theme images and icons

## Hosting
This project uses `base: './'` in Vite config, so it can be uploaded to GitHub and deployed as a static site more easily.
